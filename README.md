# INIB_AUTOMOBILE_DATA_ANALYSIS

## Project Overview

This project focuses on analyzing an automobile dataset to understand the relationship between vehicle features, pricing, fuel efficiency, and overall performance. The analysis was carried out using Python to clean the data, perform exploratory data analysis, and visualize key patterns that influence automobile characteristics and market trends.

## Objective

The main objective of this project is to perform Exploratory Data Analysis (EDA) on automobile data and identify important insights related to car prices, fuel type, engine specifications, horsepower, mileage, body style, and other vehicle attributes.

## Tools & Technologies Used

* Python
  
* Pandas
  
* NumPy
  
* Matplotlib
  
* Seaborn
  
* Jupyter Notebook
  

## Key Analysis Performed


* Data cleaning and preprocessing of automobile data
  
* Analysis of car prices across different brands and body styles
  
* Study of fuel type, aspiration, and engine characteristics
  
* Relationship analysis between horsepower, engine size, and price
  
* Mileage analysis based on fuel type and vehicle specifications
  
* Distribution and comparison of key automobile features
  
* Correlation analysis between numerical variables
  
* Data visualization for trend identification and pattern discovery
  

## Key Insights


* Identified the factors that have the strongest impact on automobile prices
  
* Compared vehicle performance based on engine size, horsepower, and fuel efficiency
  
* Analyzed pricing differences across brands, body styles, and fuel types
  
* Examined how technical specifications influence mileage and market value
* 
* Highlighted overall trends and feature relationships within the automobile dataset
  

## Conclusion

This Automobile Data Analysis project demonstrates how data analytics can be used to extract meaningful business and product insights from automobile data. It provides a clear understanding of pricing patterns, feature relationships, and vehicle performance trends through effective data cleaning, exploration, and visualization.
